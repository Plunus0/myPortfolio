<Math클래스>
java.lang에 속해있는 클래스
기본적으로 산술연산을 수행하는 메서드들을 제공한다.
모든 멤버메서드는 static으로 정의되어 있어서 객체를 생성할 필요가 없다.
정수와 실수타입의 메서드를 제공하는데 대부분 상수 double타입에 대한 주요 메서드 학습

<Math클래스의 주요 메서드>
Math.abs(x) : x의 절대값을 구한다.
Math.ceil(x) :
Math.max(a, b) : a와, b 중 최대값을 반환
Math.min(a, b) : a와, b 중 최소값을 반환
Math.random( ) : 0.0과 1.0사이의 임의의 double값을 반환
*0.0<=Math.random()<1.0
* 활용방법
1. 각 변에 n을 곱한다.(n=원하는 개별값의 갯수)
2. 각 변을 int형으로 변환한다.(소수점 이하 버림)
3. 각 변에 최저로 얻을 값을 더한다.
Math.round(x) : 실수를 소수점 첫 째자리에서 반올림한 정수를 반환(필요시 10의n자리를 곱하고 다시 double타입의 10의 n자리로 나눠서 값 도출)
