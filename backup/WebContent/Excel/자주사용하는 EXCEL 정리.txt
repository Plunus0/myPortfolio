<함수>
Sumifs : 찾을값이 있는 범위, 찾을값, 합계범위
*~+if : 조건이 들어간 함수
Vlookup : 
1. 참 으로 조건 설정시 기준열의 값은 반드시 오름차순 정렬
2. 조견표 좌클릭 후(볼드해제) F9로 배열상수로 만들면 조견표가 사라져도 값 유지)
3. 기준에 함수 가능, 기준표에도 함수 가능
(파생단가 적용시 &"*" 활용)
4. 기준의 조건이 2개 이상일 경우 AND함수 혹은 파생열(텍스트 이어붙이기) 제작
5. 대표문자(?,*) 활용시 오류확인

REPT : 지정된 횟수만큼 문자 반복표시
*=Rept(반복할 문자 또는 기호, 반복횟수)
DATEDIF : 두 날짜 사이의 간격을 계산
*Datedif(빠른날짜,나중날짜,옵션(Y=년,M=월,D=일,YM=연을 무시한 개월수,MD=연과 월을 무시한 일수,YD=연을 무시한 일수))/사용자 지정 서식으로 "년" 표시
IF(AND) : 여러조건이 모두 참일 경우
IF(OR) : 여러조건 중 하나라도 참일 경우
Sumproduct : 2개 이상의 범위를 곱한 값 표현*범위 내 셀의 갯수가 같아야함
Rank.Eq : 순위 표현 함수
*동일 순위 발생시 같은 순위로 표현
Rank.Avg : 순위 표현 함수
*동일 순위 발생시 순위의 평균값으로 표현

<배열함수>
Frequencty : 빈도수 표현함수(Ctrl+Shift+Enter로 대괄호 넣으며 마무리)

<고급필터>
데이터->고급필터
1. 목록범위
2-1. 그리고 : 조건범위(맨위는 필드명)
2-2. 또는 : 다른행으로 조건범위 추가
3. 복사위치(필드명만 따로 만들 시 해당 데이터만 출력)
*다른 Sheet에 결과 추출시 다른 Sheet에서 고급필터 실행

<값 역추산 방법>
데이터->예측->가상분석->목표값 찾기
1. 수식 셀 : 최종 값이 나타날 셀
2. 찾는 값 : 목표값
3. 값을 바꿀 셀 : 최종 값이 나타날 셀
<기능>
부분합 : 정렬로 그룹화할 항목 정리 후 진행
*새로운 값으로 대치 확인
데이터통합 : 여러시트의 데이터 통합
*데이터-데이터도구-통합
파워 피벗 : 데이터-데이터모델관리-외부데이터가져오기-Access-찾아보기-내부 데이터 선택-피벗테이블 선택

<3D맵>
닷넷프레임 웍 설치 + 인터넷 연결 필수

<차트>
차트 그래프 이미지 변경 가능

<TIP>
정확한 데이터 : '=Text 라고 입력

대표문자(?,* 등등) 변경시 반드시 텍스트앞에 "~"붙이고 변경

복잡한수식 작성시 : 띄어쓰기해도 결과엔 반영이 안되므로 함수별로 띄어쓰기 하여 가독성 UP

날짜데이터 정상설정 : 변환할 데이터 선택 후 데이터-데이터도구 텍스트 나누기(3단계만 날짜로 선택/연,월,일 확인)

사진크기조절(셀사이즈에 맞게) : Alt키 누르고 사진사이즈 조절

필요행만 삽입/삭제 : 필요행의 줄 또는 윗줄 선택후 Shift 누르고 필요한만큼 드래그

일괄입력 : Ctrl+Enter

빈셀 찾아서 입력하기 : 홈-찾기 및 선택-이동옵션 빈셀체크->입력 후 Ctrl+Enter시 전체 입력

사용자 지정 셀서식 입력방법 :
양수값의표현형태(#,##0);음수값의표현형태;0값의표현형태;문자열의표현형태
y 년도
m 월
d 일
bbbb 불기년도
aaa/ddd 요일(월, 화, 수, 목, 금, 토, 일)
aaaa/dddd 요일(월요일, 화요일...)
h 시간(24시간 초과시 대괄호)
m 분
s 초
# 숫자표기(유효하지않은 숫자 생략)
0 숫자표기(유효하지않은 숫자 0으로 표기)
@ 입력된 문자를 그대로 표기
, 천단위 표기(쉼표스타일)
*( ) 나머지여백 모두 채움

셀서식 색 표현(ex : [검정])
검정, 자홍, 녹색, 노랑, 흰색, 녹청, 파랑, 빨강

이름정의 : 정의할 셀 설정 후 이름상자 변경(F3누르면 불러오기 가능)

차트의 빈셀 처리하기 : 데이터 선택 대화상자에서 숨겨진셀/빈셀을 선택 후 숨겨진셀/빈셀설정 대화상자에서 선택

열너비가 다른 표 복사하기 : 그림으로 붙여넣기

필터시 숨김값 복사안하는 방법 : 홈-찾기 및 선택-이동 옵션

셀의 데이터에 임의의 수 연산하기 : 연산할 데이터 Crtl+C 후 대상 데이터 드래그 후 선택하여붙여넣기 후 값+곱하기 연산 활용(Crtl+Alt+V)

자동고침/하이퍼링크 설정 : 파일-옵션-언어교정
*자주사용하는 단어를 간단하게 입력 후 자동고침하여 사용

잘못 입력된 데이터 찾기 : 데이터유효성검사-잘못된 데이터 메뉴 사용 시 잘못 입력된 데이터 찾는것이 가능
*입력된 데이터에 데이터유효성검사로 입력조건을 설정한 후 잘못된 데이터 적용

숫자 앞에 0넣기 : 셀서식-사용자지정-표시형식"000"

가운데맞춤 숫자데이터 자릿수 맞추기 : 셀서식-사용자지정-표시형식"?"입력

쉼표기호로 천단위 구분하기 : 셀서식-사용자지정-쉼표활용

사용자지정 정렬하기 : 파일-옵션-Excel옵션 대화상자의 고급 - 사용자지정목록편집에서 정렬할 순서를 목록항목에 작성 -> 데이터탭-정렬 및 필터-정렬에서 정렬 대화상자표시되면 정렬 기준과 정렬방식을 선택
